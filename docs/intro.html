Introduction¶

The purpose of this module is to implement a set of useful utilities for dealing with coherent objects in LES data (in particular from MONC).

Version History¶

Latest version is 0.3.0

New at 0.3.0

  1. Use of scikit-image library tools for object labelling and bounds.

  2. Greatly speeded up code to find and merge objects spanning the domain boundary.

  3. Hence overall speedup of label_3D_cyclic and so get_object_labels of about 2 orders of magnitude.

  4. Use of loguru.logger.

New at 0.2

  1. Complete re-structuring. Extracted from Subfilter repository.
